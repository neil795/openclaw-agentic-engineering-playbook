# UI Baseline v2

## Intent
Enterprise premium portfolio presentation with outcome-first hierarchy and restrained interaction.

## Foundation choices (PR1)
- Single accent color (`--accent: #2563EB`)
- 8px spacing grid + expanded inter-section spacing
- Strong typography scale separation (hero/section/card/body)
- Tokenized depth system (background/surface/elevated/inverse)
- Accessibility defaults:
  - visible `:focus-visible`
  - reduced-motion fallback

## Follow-on work
- PR2: section differentiation + metric-first case studies
- PR3: motion polish, keyboard/a11y hardening, quality finalization
