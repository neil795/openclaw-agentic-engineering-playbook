# Motion Blueprint (Issue #3)

## Principle
Motion must improve orientation or emphasis. If decorative only, remove it.

## Motion tiers
- **Tier A (flagship, max 2 sections):** GSAP ScrollTrigger
  - Hero sequence
  - Case-study rail/pinned sequence
- **Tier B (standard):** CSS/WAAPI reveals
  - philosophy cards
  - trust items
  - timeline entries
- **Tier C (none/minimal):** dense text sections and CTA form content

## Guardrails
- Maximum 2 pinned sequences total.
- Avoid scroll hijacking.
- Avoid heavy parallax on mobile.
- Respect `prefers-reduced-motion`; all animated sections must have static equivalent.

## Section-level intent
- Hero: calm authority entrance.
- Problem landscape: metric reveal, controlled tension.
- Case studies: focused progression through intervention proof.
- AI governance: analytical, incident-report framing.
- CTA: one concise emphasis moment, no theatrics.
